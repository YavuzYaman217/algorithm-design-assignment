Insertion Sort Test Cases
==========================

Test Case 1: Small unsorted array
Input: [1, 5, 0, 1]
Expected Output: [0, 1, 1, 5]

Test Case 2: Already sorted
Input: [1, 2, 3, 4, 5]
Expected Output: [1, 2, 3, 4, 5]

Test Case 3: Reverse sorted
Input: [5, 4, 3, 2, 1]
Expected Output: [1, 2, 3, 4, 5]

Test Case 4: Array with duplicates
Input: [3, 3, 1, 2, 3, 1]
Expected Output: [1, 1, 2, 3, 3, 3]

Test Case 5: Single element
Input: [42]
Expected Output: [42]

Test Case 6: Two elements
Input: [2, 1]
Expected Output: [1, 2]

Test Case 7: Negative numbers
Input: [-5, 3, -2, 8, -10]
Expected Output: [-10, -5, -2, 3, 8]

Test Case 8: Mixed positive and negative
Input: [0, -1, 5, -3, 2]
Expected Output: [-3, -1, 0, 2, 5]

Test Case 9: All same elements
Input: [7, 7, 7, 7]
Expected Output: [7, 7, 7, 7]

Test Case 10: Large random array
Input: [64, 34, 25, 12, 22, 11, 90]
Expected Output: [11, 12, 22, 25, 34, 64, 90]
