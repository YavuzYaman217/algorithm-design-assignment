Merge Sort Test Cases
======================

Test Case 1: Small array
Input: [1, 3, 0, 7]
Expected Output: [0, 1, 3, 7]

Test Case 2: Already sorted
Input: [1, 2, 3, 4, 5]
Expected Output: [1, 2, 3, 4, 5]

Test Case 3: Reverse sorted
Input: [5, 4, 3, 2, 1]
Expected Output: [1, 2, 3, 4, 5]

Test Case 4: Array with duplicates
Input: [3, 1, 4, 1, 5, 9, 2, 6, 5, 3]
Expected Output: [1, 1, 2, 3, 3, 4, 5, 5, 6, 9]

Test Case 5: Single element
Input: [42]
Expected Output: [42]

Test Case 6: Two elements
Input: [2, 1]
Expected Output: [1, 2]

Test Case 7: Power of 2 length
Input: [64, 34, 25, 12, 22, 11, 90, 88]
Expected Output: [11, 12, 22, 25, 34, 64, 88, 90]

Test Case 8: Negative numbers
Input: [-5, 3, -2, 8, -10, 1]
Expected Output: [-10, -5, -2, 1, 3, 8]

Test Case 9: All same elements
Input: [7, 7, 7, 7, 7]
Expected Output: [7, 7, 7, 7, 7]

Test Case 10: Large random array
Input: [38, 27, 43, 3, 9, 82, 10]
Expected Output: [3, 9, 10, 27, 38, 43, 82]
