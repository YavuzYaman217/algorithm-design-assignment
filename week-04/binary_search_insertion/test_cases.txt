Binary Search Insertion Sort Test Cases
=========================================

Test Case 1: Random unsorted array
Input: [37, 23, 0, 17, 12, 72, 31, 46, 100, 88, 54]
Expected Output: [0, 12, 17, 23, 31, 37, 46, 54, 72, 88, 100]

Test Case 2: Already sorted array
Input: [1, 2, 3, 4, 5]
Expected Output: [1, 2, 3, 4, 5]

Test Case 3: Reverse sorted array
Input: [5, 4, 3, 2, 1]
Expected Output: [1, 2, 3, 4, 5]

Test Case 4: Array with duplicates
Input: [3, 1, 4, 1, 5, 9, 2, 6, 5]
Expected Output: [1, 1, 2, 3, 4, 5, 5, 6, 9]

Test Case 5: Single element
Input: [42]
Expected Output: [42]

Test Case 6: Two elements
Input: [2, 1]
Expected Output: [1, 2]

Test Case 7: All same elements
Input: [7, 7, 7, 7]
Expected Output: [7, 7, 7, 7]

Test Case 8: Negative numbers
Input: [-5, 3, -2, 8, -10, 1]
Expected Output: [-10, -5, -2, 1, 3, 8]
